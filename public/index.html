VEXTRIS

Vextris is a classic falling-block game with magic spells.

Vex glyphs ✦ appear on random blocks as they fall. Lock two marked cells beside each other in any orientation to earn a spell.

SPELLS

◆ Color Vex

Clears every cell of one random color.

Select with C, cast with V

◈ Shape Vex

Clears every cell matching one random piece shape.

Select with C, cast with V

◉ Shadow Vex

Inverts the board. Empty cells become filled, and filled cells become empty.

Requires 40% board fill.

Select with C, cast with V
